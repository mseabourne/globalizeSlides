Localize What?


Internationalization

 process of designing a software application so that it can  potentially  be adapted to various languages and regions without engineering changes

Localization

adapting internationalized software for a  specific  region or language by adding  locale -specific components and translating text

Why?

Much easier to do upfront then trying to adapt a mature site or application.

Application doesn't need to be global.  Multiple languages within the USA.

Easier for editorial to write copy.

Keep literals encapsulated and out of code and html.

What to consider

How is the current culture determined?
Different fonts needed for some languages
Handling spacing and long text
Formatting dates, times, and numbers
 Russian and Polish - different rules for plural words
US vs. JP - Formatting strings with counts.  Ex. "Top 9 Reviews" vs. "9のレシピ全てを見る »"
Right to left layouts
